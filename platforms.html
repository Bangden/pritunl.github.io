Platform and Device Support

Pritunl supports all OpenVPN clients and has official clients for several devices and platforms

macOS

The official Pritunl client is fully supported on macOS including support for profile sync.

Microsoft Windows

Windows is supported by the official Pritunl client including support for profile sync.

Linux

Both a GTK based GUI client and a CLI client are available on Linux desktops and Linux servers. Both clients have support for profile sync.

Apple iOS

An OpenVPN client is available in the iOS app store that can be used to connect to a Pritunl server. The iOS client does not support profile sync.

Android

Android has an OpenVPN client available in the Google app store that can be used to connect to a Pritunl server. The Android client does not support profile sync.

Chromebook

The easiest vpn for Chrome OS. In combination with Google single sign-on users can sign in with their Google business account and download a Chromebook compatible vpn profile in just two steps.

Ubiquiti UniFi

Support for automated and redundant IPsec peering with Ubiquiti UniFi using the Pritunl Link client. Create site-to-site links from a local UniFi network to a VPC or Google Cloud network with automated failover on both sides.
